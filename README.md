# 🚀 Zero to Hero: GenAI Engineering
> **The course that takes you from "what's an LLM?" to building production-ready AI systems — without the boring parts.**

Welcome, future AI Engineer! 🎉 You're about to embark on one of the most exciting journeys in modern tech. Whether you're a web dev, a backend engineer, or someone who just completed CS50 — this course was built for YOU.

We follow the Andrew Ng teaching philosophy: clear theory, hands-on notebooks, and a progression that builds real intuition. No hand-waving. No magic. Just you, Python, and a whole lot of "ohhhh, THAT'S how it works!" moments.

## 🛠️ Setup

```bash
# Python 3.10+ recommended
pip install numpy pandas matplotlib scikit-learn torch transformers
pip install openai langchain chromadb crewai jupyter

# Launch Jupyter
jupyter notebook
```

Or use **Google Colab** — every notebook has an "Open in Colab" badge. Free GPU included! 🎮

---

## 📐 How to Use This Course

1. **Read the README** in each module folder — it gives you the theory and context
2. **Open the notebook** (`.ipynb`) and run through it cell by cell
3. **Do the exercises** at the end of each notebook — they're designed to be fun
4. **Don't skip ahead** — each module builds on the previous one
5. **Google aggressively** — that's what real engineers do 😄
